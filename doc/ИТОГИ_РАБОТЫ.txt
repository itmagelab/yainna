🎯 КАК ИСПОЛЬЗОВАТЬ:

  ВАРИАНТ 1 (САМЫЙ БЫСТРЫЙ - БЕЗ СВОИХ ФОТО):
  ──────────────────────────────────────────────────────────────
  1. Выполните: trunk serve
  2. Откройте: http://localhost:8080
  3. Сайт работает с временными фото! ✨
  4. Замените на свои фото когда будут готовы


  ВАРИАНТ 2 (С РЕКОМЕНДУЕМЫМИ НАЗВАНИЯМИ):
  ──────────────────────────────────────────────────────────────
  1. Переименуйте свои фото согласно таблице выше
  2. Положите их в соответствующие папки
  3. Откройте: static/content.yaml
  4. Замените URL (https://...) на локальные пути (/static/images/...)
  5. Выполните: trunk serve
  6. Готово! ✨


  ВАРИАНТ 3 (СО СВОИМИ НАЗВАНИЯМИ):
  ──────────────────────────────────────────────────────────────
  1. Положите фото с любыми названиями в нужные папки
  2. Откройте: static/content.yaml
  3. Найдите секцию: portfolio -> items
  4. Замените URL на пути к своим файлам
  5. Выполните: trunk serve
