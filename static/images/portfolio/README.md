# Инструкция по размещению фотографий портфолио

## Структура папок

```
portfolio/
├── makeup/     - Фотографии макияжа (категория "Визаж")
├── brows/      - Фотографии работ с бровями (категория "Брови")
├── lashes/     - Фотографии работ с ресницами (категория "Ресницы")
└── README.md   - Этот файл с инструкциями
```

## Как добавить свои фотографии

### Шаг 1: Подготовьте фотографии

Рекомендуемые параметры:
- **Формат**: JPG или PNG
- **Размер**: 800-1200px по ширине (для оптимальной загрузки)
- **Качество**: высокое, но оптимизированное для веба
- **Ориентация**: желательно квадратная или горизонтальная

### Шаг 2: Разместите фотографии в соответствующих папках

#### Для макияжа (Визаж):
Поместите фото в папку `makeup/`:
- `evening-makeup.jpg` - фото вечернего макияжа
- `wedding-makeup.jpg` - фото свадебного макияжа
- `day-makeup.jpg` - фото дневного макияжа
- `photo-makeup.jpg` - фото макияжа для фотосессии

#### Для бровей:
Поместите фото в папку `brows/`:
- `architecture.jpg` - фото архитектуры бровей
- `lamination.jpg` - фото ламинирования бровей
- `coloring.jpg` - фото окрашивания бровей

#### Для ресниц:
Поместите фото в папку `lashes/`:
- `extensions.jpg` - фото наращивания ресниц
- `lamination.jpg` - фото ламинирования ресниц

### Шаг 3: Обновите пути в content.yaml (если нужно)

Если вы назвали файлы по-другому, откройте файл `static/content.yaml` и найдите секцию `portfolio`. 
Измените значения в поле `image` на имена ваших файлов.

Например:
```yaml
- image: "/static/images/portfolio/makeup/мое-фото.jpg"  # ← ваше имя файла
  category: "Визаж"
  title: "Вечерний макияж"
  description: "Яркий образ для особого вечера"
```

## Советы по фотографиям

### Качество фото:
- ✅ Хорошее освещение
- ✅ Четкое изображение
- ✅ Фокус на работе (брови/ресницы/макияж)
- ✅ Желательно фото "до и после" или результат крупным планом

### Чего избегать:
- ❌ Размытые фотографии
- ❌ Слишком темные или пересвеченные
- ❌ Слишком большой размер файла (более 2-3 МБ)
- ❌ Очень маленькое разрешение (меньше 600px)

## Добавление дополнительных фото

Если хотите добавить больше фотографий в портфолио:

1. Добавьте файл в соответствующую папку
2. Откройте `static/content.yaml`
3. В секции `portfolio.items` добавьте новый элемент:

```yaml
- image: "/static/images/portfolio/makeup/новое-фото.jpg"
  category: "Визаж"  # или "Брови" или "Ресницы"
  title: "Название работы"
  description: "Краткое описание"
```

## Замена фото в секции "Обо мне"

Для замены фото в разделе "Обо мне":
1. Поместите свое фото в папку `static/images/`
2. Назовите его `inna-photo.jpg` (или измените путь в `content.yaml`)
3. Рекомендуемый размер: 400-600px, портретная ориентация

---

**Важно**: После добавления фотографий пересоберите проект командой `trunk build` или перезапустите сервер разработки `trunk serve`.
