# Инструкция по работе с изображениями

## Структура папок

```
images/
├── portfolio/           - Фотографии работ для галереи
│   ├── makeup/         - Фото макияжа (визаж)
│   ├── brows/          - Фото работ с бровями
│   ├── lashes/         - Фото работ с ресницами
│   └── README.md       - Подробная инструкция для портфолио
├── fonts/              - Шрифты (если нужны)
├── background-image.jpg        - Фоновое изображение для главной секции (десктоп)
├── background-image-mobile.jpg - Фоновое изображение для мобильных устройств
├── inna-photo.jpg      - ← ВАШЕ ФОТО для секции "Обо мне"
└── README.md           - Этот файл

```

## Быстрый старт

### 1. Фоновое изображение для главной секции

В проекте используются два варианта фонового изображения:
- `background-image.jpg` - для десктопа (1920x1440px, ~414KB)
- `background-image-mobile.jpg` - для мобильных устройств (800px по широкой стороне, ~121KB)

**Как создать мобильную версию при замене фонового изображения:**

```bash
# Используя встроенную утилиту macOS (sips):
sips -Z 800 static/images/background-image.jpg --out static/images/background-image-mobile.jpg

# Или используя ImageMagick (если установлен):
convert static/images/background-image.jpg -resize 800x800\> static/images/background-image-mobile.jpg

# Или используя ffmpeg (если установлен):
ffmpeg -i static/images/background-image.jpg -vf scale=800:-1 static/images/background-image-mobile.jpg
```

**Параметры фонового изображения:**
- **Десктоп**: 1920x1440px (или аналогичное соотношение)
- **Мобильная**: 800px по широкой стороне
- **Формат**: JPG
- **Качество**: 80-85% для оптимального баланса качества/размера

### 2. Добавьте свое фото в секцию "Обо мне"

Поместите свою фотографию в папку `static/images/` и назовите её:
- `inna-photo.jpg`

**Параметры:**
- Портретная ориентация (вертикальная)
- Размер: 400-600px по ширине
- Формат: JPG или PNG
- Хорошее качество, профессиональное фото

### 3. Добавьте фотографии своих работ

Откройте папку `portfolio/` и следуйте инструкциям:

#### Для макияжа (Визаж):
Папка: `portfolio/makeup/`
- `evening-makeup.jpg` - Вечерний макияж
- `wedding-makeup.jpg` - Свадебный макияж
- `day-makeup.jpg` - Дневной макияж
- `photo-makeup.jpg` - Макияж для фотосессии

#### Для бровей:
Папка: `portfolio/brows/`
- `architecture.jpg` - Архитектура бровей
- `lamination.jpg` - Ламинирование бровей
- `coloring.jpg` - Окрашивание бровей

#### Для ресниц:
Папка: `portfolio/lashes/`
- `extensions.jpg` - Наращивание ресниц
- `lamination.jpg` - Ламинирование ресниц

### 4. Если используете другие названия файлов

Откройте файл `static/content.yaml` и обновите пути к изображениям в секции `portfolio`:

```yaml
portfolio:
  items:
    - image: "/static/images/portfolio/makeup/ваше-название.jpg"  # ← измените здесь
      category: "Визаж"
      title: "Вечерний макияж"
      description: "Яркий образ для особого вечера"
```

## Требования к фотографиям

### Технические параметры:
- **Формат**: JPG (предпочтительно) или PNG
- **Размер**: 800-1200px по ширине для портфолио
- **Вес файла**: не более 2-3 МБ (оптимизируйте для веба)
- **Качество**: высокое, но оптимизированное

### Качество контента:
✅ **ХОРОШО:**
- Четкие, не размытые фотографии
- Хорошее освещение
- Фокус на результате работы
- Фото "до и после" (где уместно)
- Крупный план для деталей (брови, ресницы)

❌ **ПЛОХО:**
- Размытые или нечеткие фото
- Слишком темные или пересвеченные
- Слишком мелкий план, где не видна работа
- Огромный размер файла (замедляет сайт)

## Оптимизация изображений

Рекомендуем оптимизировать фото перед загрузкой:

### Онлайн сервисы:
- [TinyPNG](https://tinypng.com/) - сжатие без потери качества
- [Squoosh](https://squoosh.app/) - продвинутая оптимизация
- [ImageOptim](https://imageoptim.com/) - для Mac

### Идеальные параметры:
- **Для фонового изображения (десктоп)**: 1920x1440px, ~300-500 КБ
- **Для фонового изображения (мобильное)**: 800px по широкой стороне, ~100-150 КБ
- **Для портфолио**: 1000x1000px (квадрат) или 1200x800px
- **Для фото "Обо мне"**: 500x700px (портрет)
- **Качество JPG**: 80-85%
- **Вес**: 100-500 КБ

## После добавления фотографий

1. Удалите текстовые файлы-подсказки из папок portfolio
2. Пересоберите проект:
   ```bash
   trunk build
   ```
   или перезапустите сервер разработки:
   ```bash
   trunk serve
   ```

## Добавление новых фотографий в портфолио

Чтобы добавить больше работ в галерею:

1. Добавьте фото в соответствующую папку (`makeup/`, `brows/` или `lashes/`)
2. Откройте `static/content.yaml`
3. Найдите секцию `portfolio.items`
4. Добавьте новую запись:

```yaml
- image: "/static/images/portfolio/makeup/новая-работа.jpg"
  category: "Визаж"  # или "Брови", "Ресницы"
  title: "Название работы"
  description: "Краткое описание"
```
